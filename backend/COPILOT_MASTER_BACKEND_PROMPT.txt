SKILL UPDATE LOGIC

Implement skill_service.py.

Rules:

If submission passed:
delta = problem.mastery_weight

If failed:
delta = - problem.mastery_weight * 0.2

For each skill in problem.skills_tested:

Increment total_attempts
Increment total_success if passed
Update current_mastery += delta

If mastery crosses threshold:
increase super_band

Skill update must be atomic inside DB transaction.
